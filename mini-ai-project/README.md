Mini AI Pipeline — Sentiment Classification
Author: Nur Baishah Binti Muhammad Fakhri

This project builds a small machine learning pipeline using a Bag-of-Words representation and Multinomial Naive Bayes.
The dataset is the NLTK movie_reviews corpus (link in Appendix of the report).

How to runS

Install Python 3.9+

Install packages:

pip install -r requirements.txt


Open the notebook:

jupyter notebook pipeline_demo.ipynb
